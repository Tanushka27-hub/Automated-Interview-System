# Automated Interview System

The Automated Interview System is a web-based recruitment automation platform developed to simplify the initial stages of the hiring process. The application combines WhatsApp communication, automated voice calls, interview scheduling, and candidate management into a single system, reducing the manual effort involved in coordinating and conducting preliminary interviews.

The system allows recruiters to upload candidate information, schedule interview slots, and initiate automated communication workflows. Candidates receive notifications and participate in voice-based interviews through an IVR system powered by Twilio. Their responses are collected and stored for further review through an administrative dashboard.

Built using Node.js, Express.js, MongoDB, and Twilio, the project follows a full-stack architecture with a REST API backend and a browser-based management interface. MongoDB is used for storing candidate and interview data, while Ngrok enables webhook communication during development and testing.

A major focus of the project was creating a smooth workflow that starts from candidate registration and continues through interview scheduling, automated calling, response collection, and result management. The dashboard provides recruiters with a centralized view of ongoing interviews and candidate activity, making the recruitment process more organized and efficient.

Developing this project provided practical experience with backend development, database management, REST APIs, third-party service integration, webhook handling, scheduling systems, and full-stack application design. It also offered exposure to real-world automation challenges involving communication platforms and workflow management.

### Technologies

Node.js, Express.js, MongoDB, Mongoose, Twilio, HTML, CSS, JavaScript, Ngrok, Node Schedule, Moment Timezone.

### Future Enhancements

The project can be extended with AI-based candidate evaluation, speech-to-text processing, automated scoring, analytics dashboards, and cloud deployment capabilities to support larger-scale recruitment operations.
